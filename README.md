# Bad Words Linter

![Screenshot](./screenshot.png)

## Usage

Whit [coc.nvim]():

```json
"languageserver": {
  "bad-words-linter": {
    "command": "bad-words-linter",
    "args": ["--stdio"],
    "filetypes": ["text"]
  }
}
```
